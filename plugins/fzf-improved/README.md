# fzf-improved

Custom oh-my-zsh plugin built on top of the [oh-my-zsh `fzf` plugin](https://github.com/ohmyzsh/ohmyzsh/tree/master/plugins/fzf).
Configures [fzf](https://github.com/junegunn/fzf) with rich previews, smart directory exclusions,
and `fd`-backed search.

**Load order matters** ‚Äî `fzf-improved` must come before `fzf` so its environment variables are set
before fzf's shell integration is sourced:

```zsh
# system/bundles.sh
(( $+commands[fzf] )) && plugins+=(fzf-improved fzf)
```

---

## Dependencies

| Tool | Role | Required |
|------|------|:--------:|
| [fzf](https://github.com/junegunn/fzf) | Fuzzy finder engine | ‚úì |
| [fd](https://github.com/sharkdp/fd) | Fast file and directory traversal | ‚úì |
| [bat](https://github.com/sharkdp/bat) | Syntax-highlighted file previews | optional |
| [lsd](https://github.com/lsd-rs/lsd) | Rich directory tree previews | optional |

`bat` falls back to `cat` and `lsd` falls back to `ls -lh --color=always` when not installed.

## Keyboard shortcuts

### Command line

| Linux | MacOS | Action |
|----------|-------|--------|
| `Ctrl+T` | `Ctrl+T` | Find files under the current directory and paste the selection onto the command line |
| `Ctrl+R` | `Ctrl+R` | Search command history and paste the selected entry onto the command line |
| `Alt+C` | `Esc+C` | Find a directory and `cd` into it |

Each shortcut opens an fzf panel with a live preview. See [Configuration](#configuration) for

## Tab completion

Typing `**` before `Tab` triggers fuzzy completion for any command that accepts file or directory
arguments. The candidate list is generated by `_fzf_compgen_path` and `_fzf_compgen_dir` (defined
by this plugin), which use `fd --hidden --no-ignore` ‚Äî dotfiles, git-ignored files, and directories
in `FZF_EXCLUDED_DIRS` are all included, since tab completion is typically used to target a specific
known path.

| Pattern | Completes |
|---------|-----------|
| `vim **<Tab>` | Files |
| `cd **<Tab>` | Directories |
| `ssh **<Tab>` | Hosts from `~/.ssh/config` and `known_hosts` |
| `kill **<Tab>` | Running processes |
| `export **<Tab>` | Shell variables |
| `unset **<Tab>` | Shell variables |

## Configuration

### Display ‚Äî `FZF_DEFAULT_OPTS`

Applied globally to every fzf invocation.

| Flag | Effect |
|------|--------|
| `--height=60%` | Opens inline at 60% of terminal height instead of full-screen |
| `--layout=reverse` | Results at the top, prompt at the bottom |
| `--border` | Draws a border around the widget |

### Exclusions ‚Äî `FZF_EXCLUDED_DIRS`

Directory names that `fd` skips during all searches (`Ctrl+T`, `Alt+C`/`Esc+C`, and `**`
completion). Matching is by name at any depth ‚Äî `src/node_modules` is excluded the same as a
root-level `node_modules`.

| Category | Directories |
|----------|-------------|
| Language / build / cache | `__pycache__` `.angular` `.cache` `.mypy_cache` `.venv` `.next` `node_modules` `bower_components` `dist` `build` `out` `coverage` `target` `elm-stuff` `.cargo` `.pnp` `.pnp.js` |
| Dev tools & VCS | `.vscode` `.idea` `.git` `.hg` `.svn` `.m2` `.npm` `.nvm` `.sdkman` `terraform.tfstate.d` |
| macOS system | `.DS_Store` `.AppleDouble` `.Spotlight-V100` `.Trash` `.Trashes` `Library` |
| Linux system | `lost+found` `.proc` `.sys` `.dev` `.run` `.var` |

To add exclusions without editing this file, append in `system/environment.local.sh` (gitignored):

```zsh
FZF_EXCLUDED_DIRS+=(my-big-cache another-dir)
```

> [!IMPORTANT]
> `FZF_EXCLUDED_DIRS` must be extended before the plugin loads ‚Äî use an environment file,
> not an aliases or functions file.

### Previews

| Shortcut | Preview command | Fallback |
|----------|----------------|----------|
| `Ctrl+T` | `bat --style=numbers,changes,header,grid --color=always` | `cat` |
| `Alt+C` / `Esc+C` | `lsd -lh --tree --depth 2 --color=always` | `ls -lh --color=always` |
| `Ctrl+R` | _(none ‚Äî history entries are not file paths)_ | ‚Äî |
| `**` completion | `bat` for files, `lsd` tree for directories ‚Äî all paths, no exclusions | `cat` / `ls` |

All previews open in a right-side panel at 60% width.

## Internals

`_fzf_exclude_opts` is a global array (`typeset -ga`) built at plugin load time by expanding
`FZF_EXCLUDED_DIRS` into `--exclude <dir>` flags. It is passed directly to `fd` in the command
strings for `Ctrl+T` and `Alt+C`, and is also referenced by `_fzf_compgen_path` and
`_fzf_compgen_dir` at completion time.
